Добавление комментариев к каталогам Linux

Для добавления комментариев к каталогу удобно использовать скрытый файл .comment

1) Способ с помощью функций Bash. Добавьте в ~/.bashrc:


  what_is_it() {
    cd $1
    cat ./.comment
    cd - > /dev/null
}

tell_about() {
    what_is_it $1
}

commenting() {
    cd $1
    touch ./.comment
    nano ./.comment
    cd -
}

2. Утилита для консоли comment_tool


alias add_comment="comment_tool --comment . --from-command-line "
alias tell_comments="comment_tool --view-comment . "
