Оптимизация Windows 11 после установки в случае высокой нагрузки на систему

После установки Windows 11 можно столкнуться с нежелательным поведением системы, признаками которого являются повышенная нагрузка на процессор, торможение при воспроизведении видео, сниженная скорость обращения к диску и т.д. К счастью, оптимизацию системы можно выполнить существующими программами. Для того, чтобы быстро привести систему в порядок, понадобится следующие шаги. 1. Настройка процесса WsAppx, в редакторе реестра regedit поменять значение Start с 3 на 4 в ветке HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\AppXSvc Делаем перезагрузку. 2. Запуск утилиты ESO под админом ( https://www.softportal.com/software-41169-easy-service-optimizer.html ) в режиме Оптимально. Утилита выключит сервис Bluetooth, его нужно будет активировать при необходимости. После выключения необязательных служб перезагрузка. 3. Дополнительные оптимизации удается выполнить, применяя Win 10 Tweaker от админа. Внимание в основном пунктам Конфиденциальность, Система. 4. Загружаем наиболее совместимые драйверы с помощью IO Bit Driver Booster или другой подобной программы. Тестируем воспроизведение видео. Запускаем видеофайл. Если при воспроизведении наблюдаются проблемы, ставим K Lite Codec. Если наблюдаются тормоза в браузере для какого-нибудь сайта с видео, смотрим расширенные настройки, для Chrome это chrome://flags, ставим аппаратное ускорение, если настройка сброшена 5. Настраиваем парковку процессора (можно выполнить слиянием файла реестра), сохранить в файл с расширением reg: Затем выставляем оптимальные настройки параметров процессора с помощью утилиты QuickCPU. После добавления настроек в реестр, парковку подстроить можно с помощью команд control, электропитание, настройка схемы и далее до пункта Изменить дополнительные параметры питания, смотрим пункты число ядер в состоянии простоя, Разрешить состояние простоя и так далее пробегаемся по опциям. Если процессор поддерживает настройку дросселирования тактов (троттлинга), то используем утилиты вроде ThrottleStop. 6. С помощью утилиты SSD Mini Tweaker выставляем оптимизации для SSD накопителя, если таковой имеется. 7. Работу с видео можно улучшить, обратившись к утилитам производителя видеокарты с официального сайта. 8. Отключаем встроенный антивирус. Отключаем службу обновлений. Отключаем индексацию файлов. Можно сделать с помощью services.msc. Заменить поиск может легкая утилита Everything В результате вышеперечисленных действий загрузка системы должна снизиться до 10-20%. Естественно, данные шаги применимы и к Windows 10.
